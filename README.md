# BTF Trace validator

Class that validates given BTF trace and write all errors to given file.txt

### Prerequisites

```
python 3.7
```

### Installing

 - Create virtual environment
 - Install requirements.txt using pip
 
 ```shell script
$ pip install requirements.txt
```

 - Run project using command.
