# Vk bot

Bot sending an reply to user message with weather parameters

### Prerequisites

```
python 3.7
```

### Installing

 - Create virtual environment
 - Install requirements.txt using pip
 
 ```shell script
$ pip install requirements.txt
```

 - Run project using file name.
